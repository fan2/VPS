### macOS
Mac OSX终端走shadowsocks代理 [#18](https://github.com/mrdulin/blog/issues/18)

使用proxy前先查看下当前的ip地址：

```shell
➜  ~ curl ip.cn
当前 IP：112.64.xxx.xx 来自：上海市 联通
```

执行:

```
➜  ~ proxy
➜  ~ curl ip.cn
当前 IP：47.89.xx.xxx 来自：香港特别行政区 阿里云
```
